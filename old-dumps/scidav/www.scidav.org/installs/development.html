On Windows

If you don't already own Visual Studio, you can install a free compiler based on GNU / GCC. I try to use mingw, but you get a more complete Unix command-line experience from cygwin. EPD seems to include a compiler that works for installing python tools, but the last time I tried, it was hard to use for anything else.
